CDG70 Vendor Briefing Checklists

Pre-Brief Checklist

    • Ensure Security Induction has been completed
    • Ensure the vendor got a tested radio for communication
    • Discuss the badging process
    • BB to escort the vendors all the time when moving in the building
    • Discuss the process if an alarm is triggered (stop and call security)

Post-Brief Checklist

    • Vendor to Handover to DCEO upon completion
    • DCEO to Deisolate if LOTO has been applied
